# Desafio dos Sábios

Jogo web em React + TypeScript + Vite com campanha matematica por fases.

## Estrutura

- `src/components/game`: home, telas e fluxo principal do jogo.
- `src/game`: progressao, dificuldade, perguntas, elenco e balanceamento.
- `public/videos`: introducoes dos chefes.

## Fluxo

1. Home visual-first com foco nos viloes.
2. Escolha do heroi.
3. Escolha da dificuldade.
4. Campanha com 3 fases: potenciacao, radiciacao e combinadas.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run test`
